OSRSFlipper 1.0.6 — Dashboard Tab Cleanup

Release type: UI cleanup patch

Included:
- Consolidated Account, Status / Logs, Maintenance, Settings, and About into one final Admin tab.
- Added Admin inner tabs:
  - Account
  - Status / Logs
  - Maintenance
  - Settings
  - About
- Consolidated Latest Flips, Item History, and Recurring Flips into one Market Data tab.
- Added Market Data inner tabs:
  - Latest Flips
  - Item History
  - Recurring Flips
- Restored the Item History search dropdown inside Market Data.
- Kept the existing Item History graph behavior.
- Reduced main dashboard tab-row clutter.
- Kept low-frequency tools at the end of the main tab row.
- Kept existing dashboard functionality intact.

Not included:
- New trading automation.
- Automatic GE trading.
- Automatic cancel/reprice actions.
- Database schema changes.
- AI behavior changes.
- Auto-refresh behavior changes.
- Private database files.
- Saved logs.
- Backups.
- Exports.
- Runtime cache files.
- .env.
- API keys.

Notes:
- Admin is intended for setup, account management, diagnostics, maintenance, settings, and about information.
- Market Data is intended for scanner/history views that are useful but do not need separate main-row tabs.
- This release is focused on dashboard organization and usability.
