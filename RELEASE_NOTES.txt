OSRSFlipper 1.0.5 — Trade Board and Slot Actions

Release type: feature patch

Included:
- Added Trade Board tab.
- Added stable one-table ranked trade recommendations.
- Added Buy Now / Overnight / Test Small / Avoid-Wait action labels.
- Added Trade Board scoring improvements.
- Added Capital Needed, Profit/1M, Fill, Confidence, Warning, and Reason columns.
- Added persistent Trade Board controls.
- Added Trade Board filters for Action, Confidence, and Fill.
- Fixed Trade Board filters so filtering happens before the visible row limit.
- Added Open Slot Actions from live RuneLite lastOffers.
- Added read-only GE slot pressure guidance.
- Added read-only Hold / Cancel / Reprice / Controlled Loss review suggestions.
- Added AI Advisor context from the Trade Board.
- Improved AI Advisor output to prioritize Trade Board recommendations.
- Kept Trade Board and Open Slot Actions manual-refresh only for stability.

Not included:
- Automatic GE trading.
- Automatic cancel/reprice actions.
- Automatic selling.
- Auto-refresh Trade Board polling.
- Private database files.
- Saved logs.
- Backups.
- Exports.
- Runtime cache files.
- .env.
- API keys.

Notes:
- Open Slot Actions are read-only guidance only.
- Controlled-loss suggestions require manual review.
- RuneLite lastOffers is used for live slot blockers so old unmatched trade history is not treated as a current GE offer.
